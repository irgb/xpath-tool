雷军启动红米Redmi独立品牌 手机行业老兵卢伟冰出任总经理

作者：

2019-01-10 18:40:02 来源：南方周末

  • 标签
  • 雷军
  • 红米Redmi

2019年1月10日，小米集团在北京召开全新独立品牌红米Redmi发布会，并发布该品牌首款产品Redmi Note 7。作为首款产品，Redmi Note 7坚持“死磕性价比”的极致理念，在同价位段产品中率先采用4800万超高像素旗舰相机，搭载主频2.2GHz的“满血版”骁龙660处理器，并提供18个月超长质保服务，售价仅999元起。

小米集团创始人、董事长兼CEO雷军表示：“红米Redmi独立品牌前身为红米手机产品序列。截至2018年第三季度，全球累计售出2.78亿台手机，被誉为‘国民手机’。红米Redmi独立运营后将专注电商市场，继续致力于打造极致性价比、超高品质的国民手机。”

雷军在发布会现场宣布，新近加入小米集团的前金立总裁卢伟冰，将担任小米集团副总裁兼红米Redmi品牌总经理。据了解，红米Redmi还在抖音上开设全球首个抖音快闪店，1月14日20时启动Redmi Note7销售专场。Redmi Note 7还将于1月10日18时开启全网预售， 1月15日10点现货首卖。

标配4800万超清相机，Redmi Note 7成千元旗舰新标杆

在Redmi Note 7发布前，雷军在微博上十分活跃，连发微博近30条，几乎每天都与网友互动，互动问题包括新品品质、外观改进、红米Redmi升级、多品牌战略等方面，大量收集网友意见。雷军还亲自上阵，拍摄抖音短视频，测试“小金刚”。

雷军发微博说，担心“小金刚”配置是绝对过虑，红米Redmi无惧任何对手，坚持死磕性价比，并称“生死看淡，不服就干！”引网友爆赞。

此次发布的Redmi Note 7，将手机性价比提升到了全新高度。在千元机上，采用旗舰手机刚刚使用的4800万像素超清相机，搭载部分旗舰手机配置的“新一代神U”——“满血版”高通骁龙660，内置4000mAh大电池，支持最高18W快充，真正将性价比做到极致。

值得一提的是，Redmi Note 7搭载的是主频最高2.2GHz的高通骁龙660 AIE处理器，而在同价位甚至更高价位的手机中，市场主流采用的是1.95GHz“降频版”骁龙660，Redmi Note7的满血配置更诠释了极致性价比的高品质追求。

同时，Redmi Note 7后置主摄拥有4800万超高像素，1/2英寸超大感光尺寸。Redmi Note 7相机通过像素四合一技术，输出1.6μm大像素的1200万照片，充分发挥大底传感器优势，使相机拍照细节更清晰。夜晚拍照时，1.6μm大像素进光量相比普通相机得到极大提升，暗光拍照亮度更高也更清楚。在相机功能上还支持全球 DxOMark 相机评测世界前三的小米MIX 3的“手持超级夜景功能”，这项功能将在Redmi Note 7发布两周后进行OTA升级。

Redmi Note 7不仅性能强大，而且颜值很高，设计紧跟潮流，使用6.3英寸潮流水滴全面屏，双面2.5D玻璃机身采用光影幻彩渐变设计，有暮光金、梦幻蓝及非渐变亮黑色等机身颜色，让消费者体验到随光而变、流光溢彩、如梦如幻的机身观感。

小米开启双品牌时代，卢伟冰掌舵红米Redmi

小米宣布成立独立品牌红米Redmi主攻电商市场，意味着“小米+红米Redmi”双品牌时代的到。红米Redmi与小米的关系，两者既是同一集团下的兄弟品牌，也是同一市场中的竞争对手。

值得注意的是，新近加入小米集团的前金立总裁卢伟冰将担任小米集团副总裁兼红米Redmi品牌总经理。卢伟冰在手机行业拥有二十年的从业经验，此前负责金立海外业务，有着丰富的手机产品研发、海外操盘及供应链管理经验。此次负责红米Redmi业务，有助于红米Redmi在中国区与国际市场双赛道实现快速增长。

卢伟冰表示，红米Redmi将继续追求极致性价比，主攻电商市场，吸引互联网人群。极致价比将是红米Redmi品牌始终致力追求的目标，无惧任何对手的底气，源自小米多年来对研发的重视和对供应链的把控。红米Redmi团队坚持两条腿走路，一方面盯着研发设计，一方面还要提升供应链厂商的质和量，让全球米粉拿到手上的一定是最具性价比的产品。

18个月超长质保“小金刚”，重新定义手机质量标准

据了解，“小金刚”是Redmi Note 7的内部代号，在网上热播的测试短视频中，“小金刚”在经过高跟鞋反复踩踏后依旧安然无恙，其超高品质令人惊叹。

Redmi Note 7的超强品质背后，是小米集团的不懈追求和持续投入。雷军现场展示了Redmi Note 7经历的旗舰级 “变态测试”。在用料方面，Redmi Note 7更是严格筛选供应商，如采用康宁第五代0.7mm大猩猩加0.1mm防爆层的加厚玻璃，进一步提升玻璃强度。

在此基础上，雷军宣布Redmi Note 7将打破行业12月质保的惯例，支持18个月超长质保。业内人士表示，敢于打破行业质保惯例，充分体现其对产品品质的信心。这对于一家成立仅仅8年的公司来说是难能可贵的。雷军所定义的手机质量新标准，将会掀起新一轮的手机质量革命，惠及更多消费者。

小米方面表示，品质不仅包括产品质量、服务品质，还包括用户体验与设计美学。小米要把品质提升贯穿到产品生产到消费者体验的每个环节。

雷军在发布会现场透露了一个令人惊讶的数字，反映出小米对于质量提升工作的重视程度：“仅仅2018年就召开了245次质量讨论会，从定义、设计、用料、研发、测试、售后等多个环节死磕质量。”

在生产环节中，小米不惜代价提高手机品质。红米Redmi手机出厂要经过多项品质检测，与小米旗舰系列同等级标准。小米严格筛选供应商，70%以上供应商与苹果供应商重叠，针对每款机型持续品质升级。

产品最终要交付到消费者到手中，对用户体验和服务的重视同样是品质的一环。小米通过全生命周期质量管理方式对其进行品质管控，以手机产品为例，即使产品已停止销售，小米内部也要求对其进行三年的软件版本维护。小米手机2的MIUI升级，小米维护了接近7年。同时小米要求每一代产品以一定的改进率去提升质量，明确下来并不断督促，将品质做成业界标杆。

小米在品质提升方面的不懈努力最终获得了广泛认可，2018年小米作为首家互联网公司与中广核集团、中国运载火箭技术研究院等共同获得中国质量协会颁发的质量技术一等奖，并收获了“2018年全国质量标杆”等一系列品质殊荣。

雷军曾多次提及，“创新决定我们飞得有多高，品质决定我们走得有多远”。从创立之初，小米公司就将质量工作放在核心位置，一直秉承用望远镜看创新，用显微镜看品质，坚持和用户交朋友，坚持货真价实，做感动人心的产品，让全球每个人都能享受科技带来的美好生活。
